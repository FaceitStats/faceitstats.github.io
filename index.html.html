What is it?

Faceit Stats is a minimalist bot which will give you some interesting statistics about Faceit accounts. It is very easy to use, there aren't ten thousand commands you need to learn. It is pretty and clean.

Commands

Commands Utility
.help Explicits all available commands in case you would forget about them
.csgo (.cs) 'name' Gives you exactly what you want
